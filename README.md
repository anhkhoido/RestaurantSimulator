<p>This Java project produced with IntelliJ simulates a restaurant. It demonstrates the application of two structral design patterns: the decorator and the bridge.<p>
<p>In order to put everything in perspective, the decorator encompasses the toppings a cook adds to a customizable meal like a pizza or a Vietnamese baguette. As for the bridge, it takes the form of the colour for vegetables.</p>
